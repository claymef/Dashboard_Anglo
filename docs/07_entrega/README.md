# 07 - Entregables finales

Esta carpeta contiene los artefactos preparados para la entrega del proyecto y la exposición:

```text
docs/07_entrega/
├── README.md
├── checklist_entrega.md
├── guion_presentacion.md
└── estructura_informe_final.md
```

El informe final debe derivarse de la documentación aprobada de `docs/00_gobierno` a `docs/06_cierre`. La presentación debe resumir el problema, requisitos, arquitectura, proceso ISO/IEC/IEEE 12207, desarrollo, QA, mantenimiento, resultados y próximos pasos.

El Drive `mantenimiento-trabajo` queda como repositorio de apoyo para revisar materiales compartidos. Actualmente no contiene archivos.
