## Reglas de trabajo

- Cada responsable entrega el archivo indicado y registra la evidencia correspondiente.
- Las tareas de código deben vincularse a un commit o Pull Request.
- Las tareas de pruebas deben indicar fecha, resultado y responsable.
- Una tarea solo pasa a “Completada” cuando existe el entregable y fue revisada.
