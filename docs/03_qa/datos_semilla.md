# Datos semilla para pruebas y demostración

**Código:** QA-FIXTURE-001  
**Proceso:** Verificación, validación, documentación y configuración - ISO/IEC/IEEE 12207:2017  
**Fuente:** `src/data/seedData.js`  
**Estado:** Preparado para ejecución

Los datos son estables, repetibles y no productivos. No representan información oficial de Anglo American, no se conectan a APIs y deben mantenerse constantes durante QA.

La semilla cubre resumen, impacto, inversión, aliados y KPIs de cobertura, alcance y retorno. Cualquier modificación debe registrar versión y motivo.

**Criterios:** la aplicación carga sin errores, todas las pestañas tienen contenido, los datos están identificados como demostrativos y CP-02, CP-03, CP-09 y CP-11 se ejecutan sin fuentes externas.
