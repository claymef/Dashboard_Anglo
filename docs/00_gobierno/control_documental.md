# Control documental

**Proceso de referencia:** ISO/IEC/IEEE 12207:2017 - procesos de documentación, gestión de configuración y evaluación.

## Convención mínima

Cada documento controlado debe conservar:

- Código documental.
- Título.
- Versión.
- Estado: Borrador, En revisión, Aprobado u Obsoleto.
- Autor y responsable.
- Revisor/aprobador.
- Fecha de emisión y última revisión.
- Documentos o requisitos relacionados.
- Evidencia de revisión.

## Identificación sugerida

`[FASE]-[TIPO]-[NÚMERO]`, por ejemplo `MANT-RFC-001` o `QA-PLAN-001`.

Los archivos Markdown son la fuente versionada. DOCX y PDF son formatos de distribución generados a partir de una versión aprobada.
