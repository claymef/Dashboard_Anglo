## Requisitos previos

- Node.js y npm instalados.
- Navegador moderno con soporte WebGL.
- Acceso al repositorio.

## Instalación y ejecución

```bash
npm install
npm run dev
```

Para compilar:

```bash
npm run build
```

## Uso

1. Abrir la URL indicada por Vite.
2. Revisar el resumen inicial.
3. Cambiar entre las pestañas disponibles.
4. Explorar KPIs y mini-gráficas.
5. Usar la rotación 3D con moderación.
6. Verificar la visualización del modo oscuro cuando esté disponible.
